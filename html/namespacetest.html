Detailed Description

The functions in this namespace perform operations on 2 files: the "main" file, which is the one going through the Hermes VFD, and a comparison file (with a "_cmp" suffix) which goes through the POSIX VFD. The idea is to perfrom each test on 2 files (Hermes VFD and POSIX VFD) and then compare the results at the end with h5diff. In persistent mode, the file produced by the Hermes VFD should be exactly the same as the one produced by the POSIX VFD.
